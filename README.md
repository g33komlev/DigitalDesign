# Digital Design
Программа просит пользователя ввести строку и автоматически проверяет ввод на валидность: в случае, если проверка не пройдена, выводится сообщение с описанием ошибки и пользователя просят заново ввести строку, иначе строка распаковывается и выводится в консоли. Затем можно снова ввести новую строку.

1)Программа допускает ввод строки, состоящей только из букв, и возвращает её же:
abc -> abc

2)Программа допускает ввод букв до и после распаковочных блоков:
A2[b2[s]]CD3[e]Z -> AbssbssCDeeeZ

3)Если число повторений строки равно нулю, то строка останется в том же состоянии:
a0[b] -> ab
